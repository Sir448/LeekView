# LeekView
Timeline view (formally known as gantt view) chrome extension for the LeekDuck event page.

# How To Use
1. Clone the repo
2. Go to `chrome://extensions/`
3. Turn on developer mode at the top right
4. Click "load unpacked" at the top left
5. Select the LeekView folder
6. All done! See the new view at https://leekduck.com/events/
